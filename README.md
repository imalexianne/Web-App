# Web-App
## By IMANIRAKARAMA Alexianne
This project was generated with python3.6

## Project description
An application by which a user is abble to sign up, to login, to be stored in the database, to be able to post a pitch, navigate on others'pitches and comment on them.


## Project setup instructions
Run ng serve for a dev server. Navigate to http://http://127.0.0.1:5000/. 
The app will automatically reload if you change any of the source files.

## Technologies used
I used the terminal from ubuntu and This project was generated with Python version 3.6
